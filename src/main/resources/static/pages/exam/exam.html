Detailed Examination Protocol

🚀 1. SYSTEM REQUIREMENTS
  • Desktop Only: For optimal performance, UI rendering, and proctoring stability, please use a Desktop or Laptop.
  • Mobile devices or tablets may lack the screen real estate required for the coding terminal.
2. STRUCTURE & FULLSCREEN MODE
  • Mandatory Fullscreen: The exam takes over the full-screen upon start.
  • Total 15 Questions across 3 difficulty levels: Easy, Medium, and Hard.
  • Mandatory Review: For manual finishers, use the 'Review Answers' feature to verify all responses before the final submission.
2. MARK FOR REVIEW POLICY
  • Strict Limit: You can only mark 5 questions per difficulty level for later review.
  • Marked questions are highlighted in the Question Palette for easy access.
  • You can see a full summary of your marks before final submission.
4. AI PROCTORING & SECURITY
  • Mandatory Fullscreen: On start, the exam will automatically enter fullscreen mode and stay locked there.
  • Restricted Actions: Using Ctrl, Cmd, or any other keyboard shortcuts (Copy, Cut, Paste, F12) will lead to an immediate Proctoring Violation.
  • System (5 strikes): Tab switching, focal loss, browser minimizing, or using restricted shortcuts/keys.
  • Audio (10 strikes): Integrated voice detection monitors for background sounds.
4. NAVIGATION & SHORTCUTS
  • Use Arrow Keys (Left/Right) for fast question navigation.
  • Press 'M' key to instantly toggle 'Mark for Review'.
  • The Clear Response button removes your current answer and resets status.
SECTION V: ZERO TOLERANCE POLICY

The system is strictly monitored by AI. We are not responsible for any type of cheating or misconduct. Any attempt to circumvent the security protocols will result in immediate disqualification and a permanent ban.
